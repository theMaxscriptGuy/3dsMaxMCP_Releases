3ds Max AI Assistant

Install and use the packaged assistant inside Autodesk 3ds Max. This package includes the native UI, bridge script, and bundled MCP server executable.

Install Run one MaxScript from 3ds Max.
Open Launch from the 3ds Max action category.
Use Start the server, connect, and type commands.

1. Install Using The MaxScript

  1. Open 3ds Max.
  2. Go to Scripting > Run Script....
  3. Select install_3dsmax_ai.ms from this package folder.
  4. The script installs a macro action inside 3ds Max.
  5. Add the action to a toolbar, menu, quad menu, or hotkey.
In 3ds Max customization, look for: 
Category: 3ds Max AI Assistant
Action: Open 3ds Max AI Assistant
Tip: If the action does not appear immediately, restart 3ds Max and check the same category again.

2. Open The Tool

Run the installed action from your toolbar, menu, quad menu, or hotkey.

3ds Max AI Assistant > Open 3ds Max AI Assistant

The assistant window opens inside 3ds Max.

3. Basic Usage

  1. Confirm the MCP and Bridge ports.
  2. Click Apply Ports.
  3. Click Start Server.
  4. Choose an LLM provider: OpenAI, OpenRouter, or Anthropic.
  5. Enter your provider API key.
  6. If using OpenRouter, click Load Models and choose a model.
  7. Click Connect.
  8. Type a command and click Send.

Create a red sphere at 0,0,50

Meshy

Meshy is optional and requires your own Meshy API key.

  1. Check Enable Meshy.
  2. Enter your Meshy API key.
  3. Click Start Server.
  4. Click Connect.

Image-To-3D

  1. Click Attach Images.
  2. Select one or more .png, .jpg, or .jpeg files.
  3. Send a prompt that confirms credit spending.
Use the attached image to create a Meshy image-to-3D FBX model. I confirm spending credits.

Notes

  • You do not need to run npm.
  • The packaged server starts automatically from bin\3dsmax-mcp-server.exe.
  • API keys are entered by you and are not bundled with this package.
  • If the server does not start, check whether Windows security or antivirus is blocking the server exe.
